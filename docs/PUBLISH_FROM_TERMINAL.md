# Publish From Terminal

Use the following commands to publish this repository from a terminal:

```bash
git init
git add .
git commit -m "Initial public documentation repo"
git branch -M main
git remote add origin git@github.com:Grizaceo/graph-based-legal-ir-env-law-public.git
git push -u origin main
```

If your remote URL differs, replace it in the `git remote add origin ...` command.
